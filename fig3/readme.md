**To Generate the Data:**  

The (simulation, I don't mean to talk about how the experimental data was generated) data for 3c and d came from the main run_25_half.py in the archive of mdropm, along with the chirality_map_smoothed.py from scripts. This makes several runs, optimized for this project (they don't run very far and start right before the start moving and so forth), and their results got saved to canting_tests/ with their uuids, and that's what the data is.
